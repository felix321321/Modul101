Wie füge ich einen Link ein

Um in HTML einen Link einzufügen, muss man einen bestimmten Tag im Body verwenden. Dieser ist der "< a >" Tag welcher auch wieder mit "< /a >" geschlossen werden muss.

In diesen Tag wird ein Attribut "href" hinzugefügt, welches den Link enthällt (bspw. einen Youtube Link), das ganze könnte beispielsweise also so aussehen:

< a href="https://www.youtube.com/watch?v=dQw4w9WgXcQ"> Mein erster Link < /a >

Dies würde einen Link erzeugen welcher den Namen "Mein erster Link" hat und auf die angegebene Youtube Seite weiterleiten. Ein Beispiel ist unten aufgelistet.

Wie füge ich Bilder ein

Um in HTML ein Bild einzufügen, muss man auch einen bestimmten Tag im Body verwenden. Dieser ist der "< img >" Tag welcher nicht wieder geschlossen werden muss.

In diesen Tag wird ein Attribut "src" hinzugefügt, welches den Link enthällt (bspw. den Link zu einem Bild) oder den Pfad auf eine lokale Datei, das ganze könnte beispielsweise also so aussehen:

< img src="https://www.kindernetzwerk.de/images/glossar/Doppelhelix-DNA-2a.jpg">

Dies würde ein Bild einfügen welches auf der angegebenen Seite vorhanden ist. Ein Beispiel ist unten aufgelistet.

Ein Bild als Link verwenden

Um in HTML einen Link als Bild einzufügen, muss man als Namen des Links einfach ein Bild einfügen, dies sollte sehr einfach gehen, sofern die oberen beiden Beispiele verstanden wurden.

Das ganze könnte beispielsweise so aussehen:

< a href="https://www.youtube.com/watch?v=dQw4w9WgXcQ"> < img src="https://www.kindernetzwerk.de/images/glossar/Doppelhelix-DNA-2a.jpg"> < /a >

Dies erzeugt ein Bild welches zum angegebenen Link weiterleitet. Ein Beispiel ist unten aufgelistet.
