Wie füge ich eine Liste ein

Um in HTML eine Liste einzufügen, muss man einen bestimmten Tag im Body verwenden. Dieser ist der "< ul >" Tag welcher eine ungeordnete Liste erstellet oder "< ol >" welcher eine geordnete Liste erstellt.

In diesen Tag wird ein neuer Tag "< li >" hinzugefügt, welcher die Elemente in der Liste angibt.
