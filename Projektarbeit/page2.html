Wie füge ich eine Tabelle ein

Um in HTML eine Tabelle einzufügen, muss man einen bestimmten Tag im Body verwenden. Dieser ist der "< table >" Tag welcher auch wieder mit "< /table >" geschlossen werden muss.

In diesen Tag wird ein neuer Tag "< tr >" hinzugefügt, welcher die Reihen in der Tabelle angibt.

In diesem Tag wird einer von zwei Tags verwendet um entweder die oberste Zeile in der Tabelle hervorzuheben mit "< tr >" oder eine normale Spalte anzugeben mit "< td >".
