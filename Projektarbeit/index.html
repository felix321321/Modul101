Was ist HTML

Die Hypertext Markup Language (HTML) ist eine textbasierte Auszeichnungssprache zur Strukturierung elektronischer Dokumente wie Texte mit Hyperlinks, Bildern und anderen Inhalten.

HTML-Dokumente sind die Grundlage des World Wide Web und werden vonWebbrowsern dargestellt.

Wie ist eine HTML Seite aufgebaut

Eine HTML-Seite enthällt immer einen HTML-Tag um anzugeben welche HTML-Version verwendet wird, einen Head welcher Daten enthällt die nicht auf einer Webseite ausgegeben werden und den Body welcher den Webseiteninhalt beinhaltet.

Auserdem werden sie mit ".html" abgespeichert.

Meine erste Webseite

Unter dem Text ist ein Codebeispiel welches Links die visuelle Ausgabe beinhaltet und rechts den dazugehörigen Programmcode.

Im ersten Schritt erstellen wir einen Paragraphen welcher einfach nur "Hello World" ausgeben soll, dazu verwenden wir ein "< p >" Tag zum öffnen, schreiben unsere Nachricht und schliesen den Tag wieder mit "< /p >".

Dies erzeugt eine Webseite welche den angegebenen Text enthällt. Ein Beispiel ist unten aufgelistet.
